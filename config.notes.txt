Sample config notes for example.config.json

qbt_url
- qBittorrent WebUI base URL reachable from where this script runs.

qbt_username
- WebUI username. Leave both username and password empty only if auth bypass is configured.

qbt_password
- WebUI password.

sorted_root
- Absolute root path used to build category save paths.

ignore_categories
- If a torrent's current root category matches one of these, it is skipped.

ignore_tags
- If a torrent has any of these tags, it is skipped (case-insensitive).

preserve_subcategories
- Subcategory roots to preserve from existing category.
- To preserve all, set preserve_subcategories to the literal string "*", not ["*"].

preserve_subcategory_match_anywhere
- When true, preserve matching can come from any category segment.

month_format
- Month token in final category.
- Allowed values: M, MM, MMM, MMMM.

year_format
- Year token in final category.
- Allowed values: YY, YYYY.

unmapped_category
- Category code used when tracker host has no mapping.

require_downloaded_session
- Only process torrents that downloaded bytes in this qBittorrent session.

downloaded_session_min_bytes
- Minimum session-downloaded bytes required to process.

force_auto_tmm
- Re-enable Auto Torrent Management after setting category.

dry_run
- When true, log intended actions and make no changes.

timeout_seconds
- HTTP timeout for qBittorrent API calls.

tracker_map
- Maps tracker hosts/patterns to category codes.
- Supports exact hosts and wildcard suffixes like *.domain.example.
